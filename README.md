## 🚀 Rodando dtmoney na sua máquina

Para rodar o dtmoney na sua máquina, siga estas etapas:

```
# Clone this repository
$ git clone https://github.com/neanderdev/dtmoney.git

# Go into the repository
$ cd dtmoney

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
